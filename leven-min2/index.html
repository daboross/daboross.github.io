Here's an example post using the leven-min2 framework.

Why

I was inspired to create leven-min2 when I wanted to use Leven, but couldn't find a good theme to pair with it.

I've looked at http://bettermotherfuckingwebsite.com/ before, and it seems nice. Also, I know 0% of CSS, and this is possible to make with that level of knowledge.

Why not

This looks nice, right? Maybe.

Bootstrap and other frameworks exist for a reason - and while they're slightly heavy, a bit of weight is nice.

Also, I'm not any manner of designer. All advice and themes should be taken with a grain of salt.

Ok, I'm convinced.

To use it? Good!

You can set your Leven theme with:

leven theme daboross/leven-min
